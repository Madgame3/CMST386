Converting Temperatures:

User Input
Converted Temperature

Program Design Documention:

This program prompts the user for input, and based on his or her input and prompt choices will convert those values to another temperature measurement. After the temperature has been converted, both the original user input and the converted Temperature will be displayed in the appropriate input boxes on screen.
